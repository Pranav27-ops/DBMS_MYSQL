# DBMS MySQL Notes & Lab

A comprehensive collection of DBMS (Database Management System) notes and practical lab exercises using MySQL/MariaDB for college students.

## 📚 About This Repository

This repository contains:
- Unit-wise theory notes (exam-oriented)
- SQL query examples
- Lab practical programs
- Database schema files
- Sample data for practice

## 🛠️ Tools Used

- **Database**: MySQL 8.0+ / MariaDB 10.5+
- **Client**: MySQL Workbench / phpMyAdmin / Command Line

## 🚀 How to Run SQL Files

### Using MySQL Command Line:
```bash
# Login to MySQL
mysql -u root -p

# Run a SQL file
source path/to/file.sql

# Or from outside MySQL
mysql -u root -p database_name < file.sql
```

### Using MySQL Workbench:
1. Open MySQL Workbench
2. Connect to your server
3. File → Open SQL Script
4. Select the .sql file
5. Click Execute (⚡ icon)
